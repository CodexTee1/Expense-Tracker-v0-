A simple Flask-based expense tracker that allows users to:
- Add expenses
- Edit expenses
- Delete expenses
- Filter by date and category
- Export data to CSV

## Tech Stack
- Python
- Flask
- Jinja2
- JavaScript
- SQLite
- HTML / CSS / Tailwind

## Project Status
